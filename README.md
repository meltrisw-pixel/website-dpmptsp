# Website Resmi DPMPTSP Kabupaten Maluku Tenggara

**Deskripsi:**  
Website resmi Dinas Penanaman Modal dan Pelayanan Terpadu Satu Pintu (DPMPTSP) Kabupaten Maluku Tenggara. Website ini menyediakan informasi terkait:

- Profil DPMPTSP  
- Layanan perizinan dan penanaman modal  
- Berita dan dokumentasi kegiatan terkini  
- Panduan dan kontak helpdesk  

Website ini dirancang dengan tampilan modern, responsif, menggunakan **Tailwind CSS** dan ikon **Font Awesome**.
